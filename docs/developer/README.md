# Документация для разработчиков

Этот раздел предназначен для разработчиков, которые хотят создавать собственные GUI-клиенты или автоматизировать работу с прокси-системой PP.

Основным инструментом для интеграции является бинарный файл **pp-client** (на Windows — **pp-client.exe**). Он спроектирован так, чтобы его можно было легко использовать как «движок» (backend) для графических оболочек на любой платформе — Linux и Windows.

## Содержание

1.  **[pp-client как движок](engine.md)** — все команды, пути файлов на каждой платформе, принципы работы.
2.  **[Конфигурация клиента](config-client.md)** — детальное описание структуры JSON-файла для настройки подключения.
3.  **[Гайд по интеграции](integration.md)** — управление процессом, System Proxy, Full Tunnel, логи, обработка ошибок.

## Поддерживаемые платформы

| Платформа | Бинарник | Архитектуры |
|-----------|----------|-------------|
| Linux     | `pp-client` | amd64, arm64 |
| Windows   | `pp-client.exe` | amd64, arm64 |
