# Начало работы

**PP (Proxy Protocol)** — это современная система проксирования, разработанная для обеспечения высокой скорости и максимальной скрытности.

Основная особенность проекта — протокол `pp-fallback`, который маскирует трафик под обычные HTTP-запросы к веб-сайту. Для внешнего наблюдателя ваш сервер выглядит как безобидный блог или информационный ресурс.

## Быстрая установка

Для автоматической установки сервера на Ubuntu/Debian/CentOS выполните команду:

```bash
curl -fsSL https://raw.githubusercontent.com/vakaka1/pp/main/scripts/install-server.sh | sudo bash
```

### Что сделает скрипт:
1.  **Установит зависимости**: Nginx (для реверс-прокси), Certbot (для SSL), libcap.
2.  **Загрузит бинарники**: Ядро прокси (`pp-core`) и панель управления (`pp-web`).
3.  **Настроит систему**: Создаст пользователя `pp-server` и необходимые директории в `/etc/pp`, `/var/lib/pp` и `/var/log/pp`.
4.  **Запустит сервисы**: Зарегистрирует и запустит systemd-службы `pp-core` и `pp-web`.
5.  **Настроит безопасность**: Добавит правила в `sudoers`, чтобы веб-панель могла управлять сертификатами и перезапускать прокси без прав root.

## Первый вход

После завершения установки откройте браузер и перейдите по адресу:

**`http://IP_ВАШЕГО_СЕРВЕРА:4090`**

При первом входе система попросит вас создать учетную запись администратора.
